Produsenten av Explore OnNet.

FTP Software Inc.

FTP Software Inc. er verdens største produsent av TCP/IP software for DOS/Windows plattform.
De har helt siden TCP/IP protokollens barndom vært en av de største bidragsyterne og pådriverne når det gjelder utviklingen av den desidert mest utbredte protokollen i verden.
FTP Software Inc. har hovedkontor i Andover, Massachusetts med ca. 500 ansatte.

PC/TCP OnNet

Flaggskipet til FTP. Består av en VxD basert TCP/IP kjerne og en rekke TCP/IP applikasjoner;
FTP, TN3270, TNVT, NFS, Mail, Dialer, DHCP klient, Printserver, Ping, SNMP agent, pluss mye mer.
Enkel, brukervennlig administrasjon og installasjon. Fåes også uten applikasjonene (kun kjernen).

PC/TCP Network Software for DOS/Windows

Basert på TSR kjerne, eller samme funksjonalitet som OnNet.

PC/TCP Network Software for OS/2

TCP/IP kjerne og TCP/IP applikasjoner for OS/2. Samme applikasjoner som i DOS/Windows produktet.

Explore OnNet

FTP's Internet pakke. Ideell for alle som skal knytte seg opp mot Internet via SLIP eller PPP.
Pakken består av SLIP/PPP kjerne, WWW, Mail, Gopher, TNVT, FTP, pluss mer. Meget brukervennlig.

Explore OnNet Application

En ren Internet applikasjonspakke bestående av Mosaic WWW, Mail og Gopher klient. Kan benytte en hvilken som helst WinSocket kompatibel TCP/IP kjerne.

Services OnNet

En pakke som utelukkende består av FTP's server applikasjoner. De to serverne som ikke er inkludert i standard OnNet er NFS server og DHCP server (Dynamic Host Configuration Protocol).
Produktene fåes i 1-bruker, 5-bruker, 10-bruker versjoner, samt lisens versjoner fra 20 brukere og oppover.

Produktene markedsføres i Norge av LAN Control AS, telefon 67 56 75 55 og Heath Comm AS, telefon 55 32 33 10.
Denne siden er tilrettelagt for WWW av TelePost Communication AS
